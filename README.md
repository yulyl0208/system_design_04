# Fitness Tracker

##  Запуск

```bash
docker-compose up -d
```

##  Подключение

```
mongodb://localhost:27017
```

##  Запуск скриптов

```bash
mongo < data.js
mongo < queries.js
```

##  Коллекции

* users
* exercises
* trainings

##  Возможности

* CRUD операции
* Валидация схем
* Aggregation pipeline
